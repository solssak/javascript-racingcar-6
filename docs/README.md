## 🚀 기능 구현 목록

#### 1. 경주 참여 자동차 이름 받기

- [x] 경주 참여 자동차 이름 받기
  - [x] 여러개의 이름은 쉼표(,)로 구분할 것
  - [x] 이름은 5자 이하
        추가 조건
  - [x] 문자 앞, 뒤 공백 제거

#### 2. 자동차 경주 게임 시도 횟수 받기

- [x] 자동차 경주 게임 시도 횟수 받기

#### 3. 게임 진행

- [] 경주에 참여한 자동차 이름들 형식에 맞춰 화면에 띄우기
- [] 자동차 이름 별 간 거리 '-'로 보여주기
- [] 게임 시도 횟수만큼 게임 반복하기

#### 4. 게임 결과

- [] 가장 멀리간 자동차 이름 띄워주기
  - [] 1명일 경우 => 출력, 여러명일 경우 => 쉼표로 구분해서 출력
